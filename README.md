# Bluetooth RC Car using Arduino Uno

## Overview
This project is a Bluetooth controlled RC car built using Arduino Uno and L298N motor driver. The robot can move forward, backward, left, right, and diagonal directions using a mobile application.

## Components Used
- Arduino Uno
- L298N Motor Driver
- Bluetooth Module
- DC Motors
- Robot Chassis
- Battery Pack

## Features
- Wireless control through Bluetooth
- Differential drive movement
- Smooth directional control
- Simple Arduino code structure

## Folder Structure
- `code/` → Arduino IDE code
- `circuit/` → Fritzing files and circuit design
- `images/` → Robot photos and screenshots

## How to Use
1. Upload the Arduino code to Arduino Uno
2. Connect the Bluetooth module
3. Pair mobile phone with Bluetooth module
4. Open the RC controller app
5. Send movement commands to control the car
